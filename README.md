# ApiRestNetCore
Exemplo Api Rest com Asp .Net Core usando Swagger
<a name="frameworks-supported"></a>
## Frameworks suportados
- .NET 2.0

<a name="dependencies"></a>
## Dependências
- Swashbuckle.AspNetCore
- Code First Migration (para criar Banco de Dados)
